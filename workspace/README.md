# Darwin-workspace

## Project Setup Instructions

Prerequisites (Ensure you have the following installed) :

- Git for version control

- Python 3.8+ (check the project requirements for the exact version)

## Setting up the Project in PyCharm

- Clone the project repository to your local machine.

- Open PyCharm and select "Open" from the "File" menu.

- Navigate to the cloned project directory and open it in PyCharm.

- In the Project tool window, locate the all the  src directories.

- Right-click on the src directory and select "Mark Directory as" from the context menu.

- In the sub-menu, select "Sources Root."

- This will mark the src directory as a source directory in PyCharm.

- Set interpreter from bottom right by choosing the correct path of cloned repository. This sets the virtual environment for current project.
