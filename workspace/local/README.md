            STEPS TO RUN LOCAL SETUP

(Make sure you are in root of folder)

- After following the project setup document, run below commands in terminal to install docker 
-   brew install docker
  - docker --version (To check if docker is installed)

- Follow instructions below to start the server
  - Open another terminal and run -> bash mock_start.sh (Will start the mock server for external services)
  - Open a new terminal and run -> bash start.sh (Will start the main server)
  - Enter your desktop password if prompted
